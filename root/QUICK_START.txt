## 🎨 Customization Guide

### Update Launch Date (Most Important!)
**File:** `coming_soon/script.js` (Line ~20-21)
```javascript
const launchDate = new Date();
launchDate.setDate(launchDate.getDate() + 30);  // ← Change this number
launchDate.setHours(12, 0, 0, 0);
```
- Change `30` to number of days until launch
- Countdown will automatically calculate

### Update Contact Info
**File:** `coming_soon/index.html`
- Search for `contact@vortexinfinite.xyz` → Replace
- Search for `+918270288569` → Replace

### Update Social Links

```html
<!-- Update these URLs -->
```

## 📈 What Happens With Emails

Emails submitted via the coming soon page are stored locally:

